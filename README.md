# Internship Portal Backend

A backend system for managing internships and job applications, built using Spring Boot and MySQL.

## Features
- User registration and login
- Create and view job/internship postings
- Apply for internships/jobs
- Manage applications
- RESTful APIs for all operations

## Tech Stack
- Java
- Spring Boot
- MySQL
- REST APIs

## Project Status
  In Progress – Currently building core backend features
